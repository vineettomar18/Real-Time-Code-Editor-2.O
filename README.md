# 🚀 Real-Time Collaborative Code Editor

A modern real-time code editor that allows multiple users to collaborate simultaneously, similar to Google Docs but for coding.

## ✨ Features

- 👥 Real-time collaboration (multiple users editing together)
- 🧠 Live cursor tracking
- 💬 In-app chat system
- ⚡ Instant code syncing using WebSockets
- 🎨 Syntax highlighting (supports multiple languages)
- 🧾 Code execution (optional)
- 🔐 Room-based access system

---

## 🛠️ Tech Stack

### Frontend
- React.js 
- JavaScript
- Tailwind CSS

### Backend
- Node.js
- Express.js
- Socket.IO (for real-time communication)

### Other Tools
- Monaco Editor (VS Code editor)
- Vite
